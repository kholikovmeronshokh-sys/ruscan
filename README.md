# РусСкан VPN

Русскоязычный сайт для проверки IP, VPN, ASN, провайдера, геолокации, WebRTC-утечек и сетевого отклика.

## Что умеет

- Показывает IP пользователя и версию IP: `IPv4` или `IPv6`
- Показывает страну, город, регион, ISP, ASN и сетевую организацию
- Делает оценку `VPN Score`, `Анонимность`, `Скорость`, `Уровень риска`
- Проверяет WebRTC-кандидаты и отмечает риск утечки
- Показывает privacy-флаги: `VPN`, `Proxy`, `Tor`, `Relay`, `Hosting`
- Имеет страницу `/documentation` с подробным объяснением всех полей
- Имеет страницу `/ai` с русскоязычным AI-помощником по теме IP/VPN
- Работает адаптивно на мобильных устройствах

## Переменные окружения

Добавь в Vercel или локально:

- `IPIFY_API_KEY`
- `IPINFO_TOKEN`
- `GROQ_API_KEY`
- `NEXT_PUBLIC_TELEGRAM_URL`

Шаблон лежит в [.env.example](/c:/Users/User/Documents/vpn/.env.example).

## Локальный запуск

```bash
npm install
npm run dev
```

Если `next dev` на Windows ведет себя нестабильно после перезагрузки страницы:

```bash
npm run preview
```

## Основные страницы

- `/` — главная проверка IP/VPN
- `/documentation` — документация по всем показателям
- `/ai` — тематический AI-помощник

## Deploy в Vercel

1. Загрузи проект в GitHub.
2. Импортируй репозиторий в Vercel.
3. Добавь переменные окружения.
4. Нажми `Deploy`.

## Важно

Не вшивай реальные API-ключи в клиентский код или публичный репозиторий. `GROQ_API_KEY`, `IPIFY_API_KEY` и `IPINFO_TOKEN` должны храниться только в переменных окружения.
